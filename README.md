# Micro-Account Landing Page

เว็บไซต์นึ้ทำหน้าที่เป็น Landing Page สำหรับนำเสนอและแนะนำโปรแกรม **Micro-Account** ซึ่งเป็นระบบบัญชีออนไลน์ที่รวดเร็ว ปลอดภัย และเหมาะสำหรับธุรกิจยุคใหม่

## 🚀 เทคโนโลยีที่ใช้
เว็บไซต์นี้ถูกสร้างขึ้นมาใหม่โดยยึดตามคำสั่งของคุณ:
- Next.js (App Router)
- Tailwind CSS v4
- TypeScript 5
- ลบไฟล์ .svg เริ่มต้นทิ้งทั้งหมดเรียบร้อย
- โครงสร้างแบบไม่ใช้โฟลเดอร์ `src/` (ย้าย app ออกมาด้านนอก)

## 🔧 การเริ่มต้นใช้งานโปรเจกต์ (Local)

รันคำสั่งดังต่อไปนี้ใน Terminal เพื่อเริ่มรัน Landing Page:

```bash
npm install
npm run dev
```

ตรวจสอบเว็บไซต์ได้ที่ [http://localhost:3000](http://localhost:3000)
