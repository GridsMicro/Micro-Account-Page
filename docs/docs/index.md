# 🚀 Micro Account

<div class="grid cards" markdown>

-   ⚡ **เร็ว ฉลาด อัตโนมัติ**

    ---
    ระบบบริหารธุรกิจ SME ครบวงจร ออกบิล บันทึกบัญชี จัดการภาษี ในที่เดียว

-   🔐 **ปลอดภัย มาตรฐาน Enterprise**

    ---
    Role-based Access Control, SSL/TLS encryption, Neon PostgreSQL Cloud

-   📊 **รายงานครบ พร้อมยื่นสรรพากร**

    ---
    รองรับ ภ.พ.30, ภ.ง.ด.3, ภ.ง.ด.53 และภาษี License 5%

-   🌏 **Deploy บน Vercel + GitHub**

    ---
    พร้อม CI/CD อัตโนมัติ สเกลได้ไม่จำกัด

</div>

## 📦 โมดูลทั้งหมดในระบบ

| โมดูล | คำอธิบาย | สถานะ |
|---|---|---|
| ใบเสนอราคา | สร้าง ส่ง และติดตามใบเสนอราคา | ✅ พร้อมใช้ |
| ใบแจ้งหนี้ | ออกใบแจ้งหนี้พร้อม QR Code ชำระเงิน | ✅ พร้อมใช้ |
| รอบบิลอัตโนมัติ | ออกบิลรายเดือน + ส่งอีเมลอัตโนมัติ | 🔧 กำลังพัฒนา |
| การชำระเงิน | บันทึกและติดตามสถานะการชำระ | ✅ พร้อมใช้ |
| ใบสำคัญจ่าย | เอกสารรองรับรายจ่ายบริษัท | ✅ พร้อมใช้ |
| สมุดรายวัน | บันทึกบัญชี Dr./Cr. ครบวงจร | ✅ พร้อมใช้ |
| คลังสินค้า | Physical, Dropship, License, Service | ✅ พร้อมใช้ |
| รายงานภาษี | E-Filing Dashboard สรรพากร | ✅ พร้อมใช้ |
| จัดการสมาชิก | เพิ่ม แก้ไข กำหนดสิทธิ์ | ✅ พร้อมใช้ |
