# 🚀 เริ่มต้นใช้งาน Micro Account

## ความต้องการของระบบ

!!! note "ความต้องการพื้นฐาน"
    - Browser: Chrome, Firefox, Edge เวอร์ชั่นล่าสุด
    - Internet connection (ระบบทำงานบน Cloud)
    - บัญชี Google Workspace (สำหรับ อัปโหลดเอกสาร)

## วิธีเข้าสู่ระบบ

1. เปิด Browser ไปที่ [https://micro-account.vercel.app](https://micro-account.vercel.app)
2. กรอก **อีเมล** และ **รหัสผ่าน** ที่ผู้ดูแลระบบมอบให้
3. สถานะ Account ต้องเป็น **Active** ถึงจะเข้าได้ (หากยังไม่ได้รับการอนุมัติ ติดต่อผู้ดูแลระบบ)

## แผนผังเมนูระบบ

```
Micro Account
├── 🏠 หน้าแรก (Dashboard)
├── 📄 ใบเสนอราคา
├── 🧾 ใบแจ้งหนี้
├── 🔁 รอบบิลอัตโนมัติ
├── 💳 การชำระเงิน
├── 📜 ใบสำคัญจ่าย
├── 📒 สมุดรายวัน
├── 📦 คลังสินค้า
├── 👥 ลูกค้า/คู่ค้า
├── 📊 รายงานภาษี
├── ⚙️ ตั้งค่าระบบ
└── [Admin]
    ├── 👤 จัดการสมาชิก
    └── 🔐 จัดการสิทธิ์
```
