# 📒 สมุดรายวันทั่วไป (General Journal)

คู่มือการบันทึกรายรับ-รายจ่ายบริษัทในสมุดรายวัน พร้อมการแนบใบเสร็จจาก Google Drive

---

## ภาพรวม

สมุดรายวัน (General Journal) คือหัวใจของระบบบัญชีครับ ทุกรายการเงินที่เข้า-ออกบริษัทจะถูกบันทึกที่นี่ก่อน แล้วค่อยถูกสรุปไปยังรายงานอื่นๆ

---

## วิธีบันทึกรายการใหม่

### เข้าหน้าบันทึก
ไปที่ **สมุดรายวัน** → คลิก **"ลงบัญชีใหม่"**

### กรอกข้อมูลหลัก

| ช่อง | กรอกอะไร | ตัวอย่าง |
|---|---|---|
| **วันที่เกิดรายการ** | วันที่จ่ายเงินจริง | 16/03/2026 |
| **เลขที่ใบเสร็จ** | เลขอ้างอิงเอกสาร (ไม่บังคับ) | TH001234 |
| **หมายเหตุ** | อธิบายสั้นๆ | ค่าส่งพัสดุลูกค้า มี.ค. 69 |

### กรอกรายการบัญชี

!!! tip "หลักง่ายๆ"
    - **บรรทัด "รายจ่าย"** → ชื่อค่าใช้จ่าย เช่น `ค่าไปรษณีย์`, `ค่าน้ำมัน`, `ค่าโทรศัพท์`
    - **บรรทัด "จ่ายออกจาก"** → แหล่งเงิน เช่น `เงินสด`, `ธนาคารกสิกรไทย`
    - ทั้งสองฝั่งต้องมียอดเท่ากัน ✅

**ตัวอย่าง: บันทึกค่าไปรษณีย์ 69 บาท (จ่ายเงินสด)**

| ชื่อบัญชี | ประเภท | จำนวน |
|---|---|---|
| ค่าไปรษณีย์และค่าขนส่ง | 💸 รายจ่าย | 69.00 |
| เงินสด | 🏦 จ่ายออกจาก | 69.00 |

### แนบใบเสร็จจาก Google Drive

1. คลิกปุ่ม **"เลือกจาก Google Drive"**
2. Login ด้วย Google Workspace ขององค์กร
3. เลือกไฟล์ใบเสร็จ (PDF หรือ รูปภาพ)
4. ระบบจะแสดงลิงค์ไฟล์พร้อมกดเปิดดูได้ทันที

### บันทึกรายการ

เมื่อยอดทั้งสองฝั่งเท่ากัน ระบบจะแสดง **✅ ยอดถูกต้อง กดบันทึกได้เลย!**
จากนั้นกดปุ่ม **"บันทึกรายการ"**

---

## ประเภทรายจ่ายที่พบบ่อย

| ประเภท | ชื่อบัญชีที่ควรใช้ |
|---|---|
| ค่าส่งพัสดุ, ค่าไปรษณีย์ | `ค่าไปรษณีย์และค่าขนส่ง` |
| ค่าน้ำมันรถ | `ค่าน้ำมันและยานพาหนะ` |
| ค่าโทรศัพท์, อินเทอร์เน็ต | `ค่าสื่อสารและโทรคมนาคม` |
| ค่าไฟ, ค่าน้ำ | `ค่าสาธารณูปโภค` |
| ค่าซอฟต์แวร์, License | `ค่าใช้จ่ายซอฟต์แวร์` |
| เงินเดือนพนักงาน | `ค่าใช้จ่ายบุคลากร` |
| ค่าโฆษณา, การตลาด | `ค่าโฆษณาและการตลาด` |

---

## ดูรายการที่บันทึกแล้ว

ไปที่ **สมุดรายวัน** (หน้าหลัก) จะเห็น:
- ตารางรายการทั้งหมดเรียงตามวันที่
- ยอดรวม **เดบิต (รายจ่าย)** vs **เครดิต (จ่ายออกจาก)**
- คลิกไอคอนเอกสารเพื่อเปิดใบเสร็จที่แนบไว้
