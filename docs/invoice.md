# 📄 ขั้นตอนการออกใบแจ้งหนี้ (Invoicing Workflow)

เพื่อให้ระบบบัญชีของคุณมีความถูกต้องและสามารถตรวจสอบย้อนหลังได้ 100% เราแนะนำให้ดำเนินการตามขั้นตอนมาตรฐานดังนี้ครับ

---

## 1. เพิ่มข้อมูลลูกค้า (The First Step)
ก่อนจะออกเอกสาร คุณต้องมีฐานข้อมูลลูกค้าที่ถูกต้องเสียก่อน เพื่อให้ที่อยู่และ Tax ID แสดงผลในใบแจ้งหนี้อย่างเป็นทางการ

*   **ไปที่เมนู:** `ลูกค้า/คู่ค้า` (Contacts)
*   **กดปุ่ม:** `+ เพิ่มคู่ค้าใหม่`
*   **ข้อมูลที่จำเป็น:**
    *   **ชื่อบริษัท:** ระบุชื่อเต็ม (เช่น บริษัท ตัวอย่าง จำกัด)
    *   **ที่อยู่:** ระบุที่อยู่จดทะเบียนสำหรับออกใบกำกับภาษี
    *   **เลขประจำตัวผู้เสียภาษี (TAX ID):** สำคัญมากสำหรับการยื่นภาษีมูลค่าเพิ่ม (VAT)
    *   **อีเมล/เบอร์โทร:** สำหรับการส่งประสานงาน

---

## 2. การสร้างใบแจ้งหนี้ (Invoice Creation)
เมื่อมีข้อมูลลูกค้าแล้ว คุณสามารถเริ่มออกใบแจ้งหนี้เพื่อตั้งค้างรับได้ทันที

*   **ไปที่เมนู:** `ใบแจ้งหนี้` (Invoices) -> `ออกใบแจ้งหนี้ใหม่`
*   **เลือกลูกค้า:** เลือกรายชื่อจาก Dropdown ข้อมูลที่อยู่จะเด้งขึ้นมาอัตโนมัติ
*   **ระบุรายละเอียด:**
    *   ระบุชื่อรายการ (เช่น ค่า License Software ประจำปี 2026)
    *   ระบุจำนวนเงินต่อหน่วย และจำนวน
*   **การตั้งค่าภาษี:**
    *   ตรวจสอบยอด **VAT 7%** ให้ถูกต้อง (ระบบจะคำนวณยอดรวมสุทธิให้ทันที)
    *   ตรวจสอบ **วันที่ครบกำหนด (Due Date)** โดยปกติจะตั้งไว้ 15-30 วันตามเครดิตที่ให้ลูกค้า

---

## 3. การบันทึกและตรวจสอบสมุดรายวัน
เมื่อคุณกด **"ออกเอกสาร"**:
1.  **Document System:** ระบบจะสร้างเลขที่ใบแจ้งหนี้ (INV-XXXX) และเก็บเข้าฐานข้อมูล
2.  **Accounting Sync:** ระบบจะไปลงบัญชีใน **"สมุดรายวันทั่วไป"** ให้เราโดยอัตโนมัติ ดังนี้:
    *   **Debit:** ลูกหนี้การค้า (ยอดรวมทั้งหมด)
    *   **Credit:** รายได้จากการขาย/บริการ (ยอดก่อนภาษี)
    *   **Credit:** ภาษีขาย (VAT 7%)

---

## 4. การพิมพ์และส่งเอกสาร
ระบบมีปุ่ม **"พิมพ์ใบแจ้งหนี้"** ที่ออกแบบมาให้สวยงามและพรีเมียมที่สุด
*   แนะนำให้เลือก `Save as PDF` จากหน้าต่าง Print ของเบราว์เซอร์
*   ส่งไฟล์ให้ลูกค้าทาง LINE, Email หรือ Slack ได้ทันที

> [!IMPORTANT]
> **Pro Tip:** การออกใบแจ้งหนี้จะเป็นการ "ตั้งรายรับค้างไว้" ในระบบบัญชี เมื่อลูกค้าจ่ายเงินจริง คุณต้องไปที่เมนู **"การชำระเงิน"** เพื่อทำการลดยอดลูกหนี้นะครับ!
